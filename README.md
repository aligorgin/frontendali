# frontendali.vercel.app

Techs :

- **[Next.js](https://nextjs.org/)**
- **[Vercel](https://vercel.com)**
- **[Tailwind CSS](https://tailwindcss.com/)**

## Running Locally

```bash
$ git clone https://github.com/aligorgin/frontendali.git
$ cd frontendali
$ yarn
$ yarn dev
```
